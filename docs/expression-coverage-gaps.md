### 2.5 寸止 / 高潮控制（当前几乎无专条）

> 现状：`得意边缘` 是挑衅口吻，**不是**寸止玩法；「停在入口」也不等于临高潮反复停压。

- [ ] 寸止（临高潮反复停压 / 撤离再推进）
- [ ] 多次寸止轮次（计数、轮间喘息、再逼近）
- [ ] 高潮否认 / 延后释放（合意「不准」到许可）
- [ ] 边缘控制体位（长时间停在入口；可与寸止配套，勿混写成同一条）
- [ ] 高潮后敏感续场 / 过载后续（寸止放开后的敏感续场）
- [ ] 强制保持体位（非束缚类；可与寸止「不准动」叠用）
- [ ] 失禁风险向体位（若进目录须极强同意与边界写法）

### 2.6 着衣 / 隔衣（表达层无专条）

> 现状：口味层有 `fabric_friction`（半隔衣触感）；姿势里仅零星「半脱衬衫」「背负隔衣→脱衣过渡」。**没有**「完全着衣」等表达层偏好条。

- [ ] 完全着衣（双方衣物基本不脱，隔布/开襟/拉链缝隙完成）
- [ ] 半着衣 / 只解关键扣（衬衫未除、裙裤半褪、内衣仍在）
- [ ] 隔衣磨蹭（布料摩擦升温，未必进入）
- [ ] 衣内手 / 衣内指（手伸进衣内，外观仍整齐）
- [ ] 掀衣不脱（掀起下摆/掀裙，整件仍穿在身上）
- [ ] 开襟 / 拉链缝隙插入焦点（着衣前提下的通路）
- [ ] 制服着衣不脱（与口味制服仪式区分：这里写姿势通路与布料阻力）
- [ ] 外出装束偷情着衣（外衣仍整齐，仅局部解开）
- [ ] 事后仍着衣狼狈整理（扣错扣、领带歪、裙摆皱）

## 五、补完优先级建议（供审核勾选）

1. **高**：接触部位/口技类（手口乳素股穴后穴足…）、内射/后射等结束语义、M 腿 / 对坐 / 怀抱坐具名条。  
2. **高**：偷情、露出、主人、女尊、荡妇、母狗（合意）话风包。  
3. **高**：寸止姿势专条 + 寸止口令 / 高潮许可制话风（用户已点名）。  
4. **高**：完全着衣 / 半着衣 / 隔衣磨蹭等着衣系姿势（用户已点名）；配套着衣维持话风。  
5. **中**：激烈专包、反差专包、祈求加深、一对多动作层。  
6. **低 / 待议**：失禁风险向、极端羞辱向——是否进表达层还是只靠口味/NTL。

## 六、强调项（必读）

### 6.1 成年边界：非「禁成人」，仅禁儿童性化

- 本仓库成人向内容**允许且预期存在**；边界是 **禁止儿童性化**（含校园等题材按成年设定处理）。  
- 勿把「质量收紧」写成「删掉成人」；补完条目同样须写明同意、成年、可停。

### 6.2 质量：手写、反同质化、禁止脚本生成

- 补完时**逐条手写**，主题独有落点；**不允许**脚本/模板批量灌条目。  
- 遵守 `docs/catalog-quality-standards.md`：表达层字数硬线、拉开句法、禁止净删降级。  
- 同组内禁止换皮复读；用户词条（如「深喉」「M腿」）也要写成可扮演的姿势/话风说明，而非关键词列表。

### 6.3 与 UI 偏好语义一致

- 目录再全，侧栏多选仍是**偏好高亮**，不是白名单锁死。  
- 未进目录的玩法/话风，扮演时默认仍可用；补目录只为「一键点成卡的特别偏好」。
